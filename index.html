kickstarter-tool by bronoman

Command line tool for Grails Plugin Kickstart with Bootstrap

This is a simple command line tool to accomplish a few things in the correct sequence: 1) build a new grails app with KickstartforBootstrap Plugin 2) include 3 domain classes for experimental purposes 3) build and run the app - all with one command!

Dependencies

Grails 2.0.0 Kickstart with Bootstrap 0.5.5

Install

command line tool under windows
